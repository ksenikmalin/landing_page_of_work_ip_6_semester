Задание

Темой проекта является веб-сервис для агентства по ремонту квартир.

В данной работе рассматривается удобная форма заказа услуги ремонта квартиры, обработка заявок, сбор статистики и работа с каталогом. Система создана для взаимодействия пользователей, менеджеров и администраторов.

Целью данного курсового проекта является разработать веб-страницы, интеграция по API с внешним сервисом (личный кабинет для 3х ролей (клиент, менеджер, администратор), дашборд, реализация CRUD, фильтрация и поиск.

Актуальностью данной темы появилась с развитием компьютерных технологий, благодаря чему возросла потребность хранения и обмена информацией. Её хранение играет важную роль в работе человека и предприятия. Ведь от достоверно правильной и доступной информации зависит успех его работы. В современном мире для хранения больших объёмов информации были разработаны базы данных (БД). Для более эффективного развития данной отрасли необходимо использовать автоматизированные информационные системы.

Для достижения данной цели были выявлены следующие задачи:

  1. Ознакомиться с литературой по данной теме и провести анализ предметной области «Агентство по ремонту квартир»;
  2. Анализ аналогичных сервисов;
  3. Проектирование макетов интерфейса «Агентство по ремонту квартир» в Figma;
  4. Разработка и тестирование веб-сервиса.

Используемые технологии


  1. CSS - формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Преимущественно используется как средство описания, оформления внешнего вида веб-страниц, написанных с помощью языков разметки HTML и XHTML, но может также применяться к любым XML-документам, например, к SVG или XUL.
  2. HTML - стандартный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами и отображается в виде документа в удобной для человека форме.
  3. Angular - фреймворк от компании Google для создания клиентских приложений. Прежде всего он нацелен на разработку SPA-решений (Single Page Application), то есть одностраничных приложений. Angular предоставляет такую функциональность, как двустороннее связывание, позволяющее динамически изменять данные в одном месте интерфейса при изменении данных модели в другом, шаблоны, маршрутизация и так далее. Одной из ключевых особенностей Angular является то, что он использует в качестве языка программирования TypeScript.
  4. Node.js - это среда выполнения кода на JavaScript, которая построена на основе движка JavaScript Chrome V8, который позволяет транслировать вызовы на языке JavaScript в машинный код. Node.js прежде всего предназначен для создания серверных приложений на языке JavaScript.
  5. Express.js - это фреймворк для Node.js, который реализовывает слой функций, необходимых для создания эффективных приложений и API. Его использование значительно сокращает написание кода, а, значит, уменьшается затрачиваемое на разработку время.
  6. MySQL - это система управления реляционными базами данных (СУБД). На сегодняшний день это одна из самых популярных систем управления базами данных.
  7. Moment.js - это библиотека, позволяющая использовать разнообразные функции для операций с датами. Moment.js была создана для обработки даты и времени и их различных сочетаний.
  8. NGX Bootstrap - это библиотека, в которой можно найти базовые компоненты, реализующие возможности шаблона Bootstrap и написанные для Angular, да и не только их. Она подходит для разработки настольных и мобильных приложений и спроектирована с учётом расширяемости и адаптивности.
  9. Webpack - один из самых мощных и гибких инструментов для сборки frontend. Отлично работает с одностраничными приложениями. Позволяет осуществлять продвинутое разделение кода. Это наиболее популярный инструмент разработки.
  10. Chart.js - это популярный инструмент, который предназначен для создания графиков и диаграмм. С помощью данного инструмента удобно реализовывать дашборды.

    11. Также для разработки веб-ресурсов использовались:
  11. Git - это набор консольных утилит, которые отслеживают и фиксируют изменения в файлах. В проекте предусмотрен контроль версий с помощью Git и онлайн-сервиса GitHub.
  12. Draw.io – бесплатное приложение, предназначенное для моделирования диаграмм и блок-схем бизнес-процессов.
  13. Figma - онлайн-сервис для разработки интерфейсов и прототипирования с возможностью организации совместной работы в режиме реального времени. Макет разрабатываемых страниц проекта выполняется в данном онлайн-сервисе.
